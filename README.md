# infinite-theme
Minimilistic Website Theme created with HTML, CSS and JS

### [Live Demo]("https://tarun-bisht.github.io/infinite-theme/")

### Features:
- Fully Responsive Website Template
- Simple frontend with HTML, CSS, Bootstrap and JQuery
- Animations Effects

### Dependencies
- ImageHover.css
- Animate.css
- JQuery
- Bootstrap
